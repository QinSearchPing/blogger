# blogger

> My kryptonian Nuxt.js project

## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```

## Remarks
```
# Server 服务器端框架
$ nuxt default server
# UI-Framework UI框架
$ Element UI
# Nuxt model Nuxt 模式
$ Universal
```
## Dependencies
```
Axios Http请求操作
EsLint 保存时代码规范和错误检查代码

```
